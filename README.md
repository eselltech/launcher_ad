# EsellBanner

#### 介绍
易信发launch广告sdk


#### 使用说明

1. 添加类库 esell_banner
2. 在布局引用 如：
```
<com.esell.esellbanner.EsellBanner
        android:id="@+id/eb_content"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:background="@color/colorAccent" />
```
3. 轮播时间自己控制，没有就是15s
4. 默认背景自己设置

#### 注意事项

1. 应用必须给与sd读写权限，没有用不了
